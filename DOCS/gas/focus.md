# Centre: Focus

## Purpose
Monthly mission map (habits, routines, additions, eliminations).

## Entry
- UI: `gas/Game_Focus_Centre.html` (inline)
- Backend: `gas/game_focus.gs`

## Storage
- Drive root: `Alpha_Game/Alpha_Focus`
- Subfolders: `Current`, `Q1`, `Q2`, `Q3`, `Q4`
- Logsheet: `Alpha_Focus_Logsheet`
- Obsidian export: `Alpha_Game/3_FOCUS`

## API (GAS)
- `saveFocusEntry(domain, mission, month, type)`

## Script Properties (UserProperties)
- `FOCUS_DRIVE_FOLDER_ID`
- `FOCUS_LOG_SHEET_ID`

## Notes
- Uses the Drive vault named `Alpha_Game` as the Obsidian root.
