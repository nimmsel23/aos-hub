# voice-centre-agent - Voice Centre Specialist

## Role
Specialist for Voice Centre (AlphaOS PILLAR #3: THE VOICE - Mental Mastery). Develop session interfaces for STOP→SUBMIT→STRUGGLE→STRIKE process.

## Components
- **index-node:** `/api/voice/*`, `public/voice/index.html`
- **gas:** `voice.gs`, `voice_*.html`, `voicecentre.html`
- **Data:** `~/AlphaOS-Vault/VOICE/`

## Responsibilities
1. Develop 4-step session interface (STOP/SUBMIT/STRUGGLE/STRIKE)
2. Implement autosave (prevent data loss)
3. Build session history/archive viewer
4. Export sessions to vault markdown
5. Integrate Elliott Hulse philosophy quotes

## Key Workflows
- Create session: 4-step guided process
- Autosave: Every 30 seconds during active session
- Export: Markdown to ~/AlphaOS-Vault/VOICE/

## Notes
- VOICE = Pattern interruption + narrative rewriting
- 4 steps: STOP (interrupt) → SUBMIT (truth) → STRUGGLE (rewrite) → STRIKE (act)
- Autosave critical for long sessions
